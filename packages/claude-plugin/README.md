# vovk

Claude Code plugin for [Vovk.ts](https://vovk.dev). Wraps `vovk-cli` with two skills so project setup and code generation go through the CLI rather than hand-edited files.

## Skills

- **`vovk:init`** — initialize Vovk.ts in a Next.js App Router project, or scaffold a fresh Next.js app and run `vovk init` on top.
- **`vovk:new`** — create segments and modules (controller + service) in an existing Vovk project.

## Install

Add this repository as a Claude Code plugin marketplace, then install the `vovk` plugin from it.
